# 🎙️ VoiceCamTranslator

一个使用 **Python** 编写的语音随机翻译器。  
您只需要对着麦克风说出一句话，程序会自动识别语音、随机选择一种语言，并将内容翻译为该语言显示出来。  

---

## 🚀 功能特点

- 🎧 **语音识别**：通过 Google Speech Recognition 识别您的语音  
- 🌍 **随机语言翻译**：使用 Google Translate 随机选择语言进行翻译  
- 💬 **简洁易用**：几行命令即可运行  
- 🔊（可选）未来计划添加语音朗读翻译结果  

---

## 🛠️ 安装方法

1. 克隆项目仓库：

   ```bash
   git clone https://github.com/createapppythonRussell/VoiceCamTranslator.git
   cd VoiceCamTranslator
   ```

```bash
pip install -r requirements.txt
```

```bash
python 摄像头.py
```
